# dw-cracow-recipy-server

## Installation

```bash
git clone https://github.com/filusn/dw-cracow-recipy-server.git
```
```bash
cd dw-cracow-recipy-server
```

```bash
python -m venv venv_dw
```

```bash
venv\Scripts\activate
```

```bash
python -m pip install -r requirements.txt
```

```bash
python recipy.py
```
